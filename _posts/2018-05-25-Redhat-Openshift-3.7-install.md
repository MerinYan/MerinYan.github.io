## 环境准备
* 服务器地址：  

| IP            | HostName            | CPU     | Memory              | Disk           | Description   |  
|:--------------|:--------------------|:--------|:--------------------|:---------------|:--------------|  
|192.168.1.102  |master.infodt.com    |8 Core   | 16 GB               | / 40GB         | Master        |  
|192.168.1.103  |node1.infodt.com     |8 Core   | 16 GB               | / 40GB         | Data Node     |  
|192.168.1.104  |node2.infodt.com     |8 Core   | 16 GB               | / 40GB         | Data Node     |  
|192.168.1.105  |inode1.infodt.com    |8 Core   | 16 GB               | / 40GB         | Infra Node    |  
|192.168.1.106  |inode2.infodt.com    |8 Core   | 16 GB               | / 40GB         | Infra Node    |  


* Selinux：必须打开
```
SELINUX=enforcing
SELINUXTYPE=targeted
```

* 系统注册
```
subscription-manager register --username=<user_name> --password=<password>
subscription-manager refresh
```

* 时钟同步

* DNS：需要能解析到所有主机，不能单靠hosts文件
```
[root@master ~]# more /etc/resolv.conf 
# Generated by NetworkManager
search 192.168.1.30 infodt.com
nameserver 192.168.1.30
nameserver 114.114.114.114
[root@master ~]# dig master.infodt.com @192.168.1.30 +short
192.168.1.102
...
[root@master ~]# dig inode2.infodt.com @192.168.1.30 +short 
192.168.1.106
```
